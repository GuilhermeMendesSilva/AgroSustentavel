# 🌱 Projeto Agrinho 2026 - Agro Forte, Futuro Sustentável

Bem-vindo ao repositório do projeto web desenvolvido para o **Programa Agrinho 2026**, no estado do Paraná. Este site tem como objetivo demonstrar como a tecnologia e a inovação estão transformando a agricultura, promovendo a produtividade em harmonia com a preservação ambiental.

---

## 🎯 Sobre o Projeto

O tema escolhido para este ano é **"Agro forte, futuro sustentável: como a tecnologia auxilia na produção"**. 

A agricultura moderna (Agricultura 4.0) utiliza ferramentas tecnológicas de ponta para otimizar recursos, evitar o desperdício e garantir o cuidado com o solo e a água. Além de apresentar essas tecnologias (como drones, IoT, Inteligência Artificial e Biotecnologia), o projeto também destaca a importância do **Protagonismo Jovem e da Sucessão Familiar** no campo.

## ✨ Funcionalidades e Design

O site foi construído com foco na **Experiência do Usuário (UX)** e em uma **Interface do Usuário (UI) minimalista e soft**, transmitindo a modernidade e a leveza necessárias para o tema da sustentabilidade.

* **Design Responsivo:** O layout se adapta perfeitamente a celulares, tablets e computadores de mesa.
* **Animações Suaves (Scroll Reveal):** Os elementos surgem suavemente na tela à medida que o usuário rola a página, garantindo uma leitura dinâmica.
* **Header Dinâmico:** O menu superior ganha um efeito de desfoque e sombra ao rolar, mantendo a facilidade de navegação sem poluir a visão do conteúdo.

## 🛠️ Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as linguagens fundamentais da web, sem o uso de frameworks externos para garantir leveza e demonstrar domínio dos conceitos base:

* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização baseada em variáveis (Custom Properties), design minimalista, Flexbox e CSS Grid.
* **JavaScript (Vanilla):** Lógica para animações através da API `IntersectionObserver` e manipulação do DOM.

## 🚀 Como Executar o Projeto

Como o site é totalmente estático (Front-end), rodar o projeto é muito simples:

1.  Faça o download ou clone este repositório para o seu computador.
2.  Abra a pasta do projeto.
3.  Dê um duplo clique no arquivo `index.html` para abri-lo diretamente no seu navegador padrão.
4.  *(Opcional)* Se estiver usando o VSCode, você pode utilizar a extensão **Live Server** para uma experiência de desenvolvimento em tempo real.
